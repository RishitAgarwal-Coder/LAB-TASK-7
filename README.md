# Student Roll Number and Queue Management Tasks

This repository contains C implementations for basic data structure operations including Queue reversal and Linked List manipulations.

**Logistics Queue Reversal:** Uses a temporary stack logic to reverse a processing sequence without affecting future arrivals.

**Singly Linked List Operations:**
- **Roll Number Management:** Inserting students' roll numbers at the end of a list.
- **List Reversal:** Populating a list by inserting at the front and then reversing the entire chain.
- **Middle Element Detection:** Efficiently finding the center of a list using slow and fast pointer traversal.

**Objective:**
Mastering fundamental pointer manipulation and LIFO/FIFO principles through real-world scenarios like package sorting and student data management.
